Oolong

A soothing dark theme for late nights.

Oolong is based on Jon Blow's Emacs custom theme and its variants (be it Sublime Text, Vim, etc).

It is built for professionals that appreciate simplicity, focusing on what matters — control flow, types, comments.

If you want support for your favorite language, or have any suggestions, please file an issue.
